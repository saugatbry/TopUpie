Watch your favourite anime anywhere, anytime. No ads, no bullshit.

**TopUpie Anime** — built with [Next.js](https://nextjs.org/), [Shadcn/ui](https://ui.shadcn.com/), and [Tailwind CSS](https://tailwindcss.com/). Data from [Jikan API v4](https://docs.api.jikan.moe/), video from [MegaPlay](https://megaplay.buzz).

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). No backend needed — auth is localStorage-based, API calls are server-side cached.

## Deploy to Vercel

1. Push to GitHub
2. Import repo at [vercel.com/new](https://vercel.com/new)
3. Vercel auto-detects Next.js — no config required
4. (Optional) Add custom domain
